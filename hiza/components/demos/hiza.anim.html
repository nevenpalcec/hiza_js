HIŽA animations

Import url:
    https://cdn.jsdelivr.net/gh/nevenpalcec/hiza_js/hiza/components/hiza.anim.css
How to use classes:
    h-anim_name-speed-type

Fade in/out

  • anim_name = 'fade_in' or 'fade_out'
  • speed = 'slow', 'fast' or empty
  • type = 'keep' or empty

div_1.classList.add('h-fade_in'); div_2.classList.add('h-fade_in-slow'); div_3.classList.add('h-fade_in-fast-keep'); div_4.classList.add('h-fade_out-keep');
